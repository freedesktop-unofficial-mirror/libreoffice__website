===========
Permissions
===========

Create the browser object we'll be using.

    >>> from Products.Five.testbrowser import Browser
    >>> browser = Browser()
    >>> browser.handleErrors = False

By default the LEC is visible to anonymous users.

    >>> browser.open(portal.lec.absolute_url())
    >>> print browser.contents
    <...
    ...Currently listing 0 projects with 0 releases...

Anonymous users can't add projects.

    >>> browser.getLink('Add Extension Project')
    Traceback (most recent call last):
    LinkNotFoundError
    >>> browser.getControl('Add New Extension Project')
    Traceback (most recent call last):
    LookupError: label 'Add New Extension Project'

Log in as a normal member.

    >>> from Products.PloneTestCase import ptc
    >>> browser.getLink('Log in').click()
    >>> browser.getControl('Login Name').value = 'member1'
    >>> browser.getControl(
    ...     'Password').value = ptc.default_password
    >>> browser.getControl('Log in').click()
    >>> print browser.contents
    <...
    ...Welcome! You are now logged in...

By default the LEC is in the open workflow state so normal members can
add projects.

    >>> browser.getControl('Add New Extension Project').click()
    >>> browser.getControl('Title').value = 'Foo Project Title'
    >>> browser.getControl(
    ...     'Project Summary').value = 'Foo Project Summary'
    >>> browser.getControl(
    ...     'Full Project Description'
    ...     ).value = 'Foo Project Description'
    >>> browser.getControl('1 - Planning').selected = True
    >>> browser.getControl('Stand-alone products').selected = True
    >>> browser.getControl(
    ...     'Contact address'
    ...     ).value = 'http://nohost/plone/lec/foo-project-title'
    >>> browser.getControl('Save').click()
    >>> print browser.contents
    <...
    ...Changes saved...

If the LEC is in the closed state, members can't add projects.

    >>> portal.portal_workflow.doActionFor(portal.lec, 'retract')
    >>> browser.open(portal.lec.absolute_url())
    >>> browser.getLink('Add Extension Project')
    Traceback (most recent call last):
    LinkNotFoundError
    >>> browser.getControl('Add New Extension Project')
    Traceback (most recent call last):
    LookupError: label 'Add New Extension Project'
