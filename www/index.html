News

November ~24th - OO.o hacking bounties.

Novell announces desktop integration and tweaking bounties here.

October ~25th - Novell add engineers.

Novell are devoting more great people to working full time on OO.o, please welcome Anil Bhatia, Jayant Madavi & Amit Shoivake as they transition to the team.

September ~30th - RedHat hire DanW.

As above really, Dan Williams will be hacking on OO.o under a Red-Hat full-time - which promises to be great news for everyone.

September 5th - More magicpoint help.

Consultations with Nat revealed the existence of the carefully concealed 'Master View' button, updated the MGP users guide.

August 5th - Updated hackers guide.

I updated the hackers guide to describe the new 'openoffice' module, and binned reams of cruft. Added a bit on going around a development iteration, and archived the old hyper-detailed explanation. Also, Ximian got bought by Novell - which is good news for OO.o development. Also, added the beginnings of a Users' FAQ with answers.

July 31st - Gnome/OO.o mailing list / Volker

It turns out Volker is a hero, and has been working tirelessly to fix the multitudinous problems thrown up by migrating to the new machine. Also; we've created a new mailing list for Gnome/OO.o, subscribe here.

July 18th - New machine

ooo.ximian.com has moved to a newer, bigger machine; this should allow lots of nice new services - eg. ViewCVS. Unfortunately there are still some troubles switching across; the old machine is available here for Bonsai.

July 17th - Flat patch list

In response to no demand at all; bleeding edge patches can now be found in patches - copy them into a working ooo-build setup to break everything.

July 17th - New RC2 snapshot & Valgrind

A new Ximianized OO.o 1.1 source set is available here with fixed native font sizing and toolbar icons, plus other fixes, red-carpet packages to follow. Also there is a new improved Valgrind available for cleaning OO.o.

July 14th - Valgrind bits

Julian sent me this tutorial on how to use Valgrind. Also, Dan Kegel has written a great piece on using Valgrind with OO.o here: Don't delay - grind that code today.

July 14th - New snapshots & User <-> Issuezilla mapping

New snapshots from the RC2 branch taken no 7th July are available via red-carpet; do grab them and start filing bugs. Also, to help people file bugs we've added a mapping of Names to IRC / Issuezilla accounts here.

June 27th - OO.o 1.1 snapshots

A snapshot build of OO.o 1.1 is available for SuSE 8.2 and Redhat 9.0 via. Ximian red-carpet. We hope to have more regular snapshots available in a week or so.

June 20th - GUADEC slides

More slides got manufactured for GUADEC they are available here, we also had lots of interest, and a good number of people signed the JCA 'today'.
