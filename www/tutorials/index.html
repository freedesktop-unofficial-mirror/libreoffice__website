OO.o hackers tutorials

Since there are a number of common tasks that people want to get done, and looking at these in detail can be instructive generally - we've provided a set of annotated patches to help people get stuck into the code.

Unfortunately 'Toolbars' is obsolete as written since it had targetted OO.o 1.1.x.

  • Accelerators (diff) - adding / changing the default K/B accelerators.
  • Toolbars (diff) - adding / changing the default toolbar items.
  • Simple dialogs (diff) - adding items to & altering the about dialog
  • More dialogs (diff) - adding a widget & i18n string & callback.
  • Writer Context Menu (diff) - adding a new menu command & it's implementation.
  • Impress Context Menu (diff) - adding a new menu command & it's implementation.
  • Build infrastructure (diff) - creating a skeleton project / toplevel.
  • Creating a new library (diff) - Making up a new library of the toplevel project.
  • Using the new library (diff) - Using the library from a different toplevel project.

You know ? :-D

  • UNO-fying the new library (diff) - Making a UNO component out of the new toplevel project.
  • Specifying own idl (diff) - Creating an own idl file for our new UNO component to implement.
