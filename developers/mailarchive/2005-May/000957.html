	Quite sometime since my last status report. I've been looking into the
VBA interoperability all these days. Lot of code reading, notes etc.
Got some initial interfaces defined and Michael's helped with an initial
implementation/stub and integrating it into the build system. He has the
process of adding new interfaces captured in hackers guide.

Interfaces for WorkBook, WorkSheet and Range are defined and some dummy
implementation available for these.

the relation ship between WorkBook, WorkSheet and Range in .idl, as it

Similar is the case with Range to WorkSheet.
