Hi Ruslan,

The problem I had seen there was that there was a variable "$(OUTDIR)"
in the db_java project file that devenv could not migrate/convert to
"Release" in the newer project file that it tries to. The patch attached
is what I had used to get over it, please check if it is the same
issue.

Regards,
