Hi,

One Unix ICU is used, but only for some layouts, those that require
complex text layout.  There are cases where a simple, non-ICU text
layout system is used (derived from teh 1.0 series builds), but most stuff
goes through ICU right now.  I would be that anything Indic would have to
go through the ICU layout engine.

Windows is completely different, and does _not_ use ICU.  Michael is
correct, it uses the Uniscribe APIs from Microsoft, but again only for
really complex stuff.  There is still code that does less-complex text
layout that doesn't use Uniscribe and is essentially the older code from
the 1.0 series builds.

Dan
