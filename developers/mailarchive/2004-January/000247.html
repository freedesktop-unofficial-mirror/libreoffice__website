Hi,

Martin is correct, RHFedoraOnly will only apply a subset of the patches
required.  There is also a RHBase (?) item that applies patches that all
Red Hat distributions will use.  However, RHFedoraOnly won't pull those
patches in.  To get a full Fedora version, use RHFedora.

Sorry if this was unclear.

Dan

> If you compiled --with-distro=RHFedoraOnly, then that's the reason for the bad
> background color. With that switch, you get only a very small (and odd) set of
> patches: those that are used only for Fedora. This means the patches common to
> all distros won't be applied. Some of the common patches make the icons look
> better :). Safe values for --with-distro are Minimal, Ximian, and RHFedora.
> The other patch sets were probably never built with ooo-build.
