Hi,

Yes, since ooo-build-derived OpenOffice.org installations do not use the
"spadmin" program, and hence the libspa* libraries, these will not be
installed.  This is normal.  All font configuration is done with normal
Linux font tools, and fonts are located from fontconfig places.

What we should do is to patch the *.scp files that install these libraries
so the warnings go away.

Dan
