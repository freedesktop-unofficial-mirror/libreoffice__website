Hi everyone,

We have a total killer bug in Gentoo with ooo-build-2.0.0: When building
whithout java, saving in the standard formats like odt or sxw does not
work anymore. Instead of saving it just pops up the file dialog again. I
definately is not a file selector problem, as it happens with both the
original one and the gnome file selector. Also the file type
descriptions are not properly listed when trying to manually select a
file type. See:

http://bugs.gentoo.org/show_bug.cgi?id=110131

for all the gory details. This does not happen with the original binary
from OOo.

So anyone else seen this? Or is this even expected behaviour now?
