OpenOffice.org build:

	This package contains Desktop integration work for
OpenOffice.org, several back-ported features & speedups, and a much
simplified build wrapper, making an OO.o build / install possible for
the common man. It is a staging ground for up-streaming patches to
stock OO.o.

	This is the first release from the new stable ooo-build-2-0 branch
targetting OpenOffice.org 2.0. Longer term development continues on
ooo-build HEAD.

Changes in this minor release:

    + features
	+ target 2.0.0
	+ updated artwork (Jimmac,Petr,Guiseppe)
    + build fixes
	+ Win32 (David, Michael)
	    + enable crystal/industrial icons
	    + fix linux-specific system-bits
	    + gengal rename
	+ PPC stlport fix (Andreas)
	+ libwpd / srcdir (Andreas)

Package available from:
       
http://go-oo.org/packages/OOO680/ooo-build-2.0.0.tar.gz

	To find out more and/or get involved in OOo hacking see
http://go-oo.org/wiki and/or subscribe to our OO.o development
mailing list at:
        http://lists.ximian.com/mailman/listinfo/openoffice.
