Hi,

I have just added the Tango icon theme to the ooo-build-2-1 branch. It will be 
used in GNOME and Win32 by default.

Please let me know if you disagree with it. I could make it Novell only...


Do not forget to download 
http://go-oo.org/packages/SRC680/ooo_tango_images-1.tar.bz2 and install it 
bellow ooo-build/src.

In the incremental build, you would need to remove scp2/unxlng*. There are 
somewhat broken dependencies, so the icon theme will not be installed 
otherwise.

I am going to add it also to the ooo-build trunk. Note that the Tango icon 
sources are already included in OOo-2.2, so the ooo_tango_images-1.tar.bz2 
will not be necessary there.
