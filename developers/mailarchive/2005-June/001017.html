We use system myspell dictionaries installed in /usr/share/myspell. They are 
used by Mozilla Composer as well. We have a lot of myspell-<lang> packages 
based on the sources from 
http://lingucomponent.openoffice.org/spell_dic.html. There are much more 
dictionaries than in the OOo sources.

The script /usr/lib/ooo-2.0/install-dict is able to update symlinks 
from /usr/share/myspell to /usr/lib/ooo-2.0/share/dict/ooo and 
update /usr/lib/ooo-2.0/share/dict/ooo/dictionary.lst. So, I call this script 
from triggers in the myspell-* packages after OOo is installed.
