Hi,

Up-stream now splits the source archives the same way we do.

I've updated ooo-build to use those by default when you use the ./download 
script.  It will get them from a 'Bouncer' mirror, or from 
http://www.go-oo.org/packages/upstream-copy/ depending on your --with-mirror= 
settings:
- --without-mirror => Bouncer
- --with-mirror=blah -> "blah/upstream-copy/"
