True, but not applicable here: This whole thing is in #ifdef __cplusplus 
anyway, because C compilers barf on

char Min(char a, char b) { something }

as well -- C doesn't allow function distinction through the arguments only -- 
it's just replacing C++ specific code with C++ specific code.
