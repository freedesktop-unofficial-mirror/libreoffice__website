	The trick is to edit the patch; a patch looks like this:

--- filea
+++ fileb
@@ -<filea-start>,lines +<fileb-start>,lines
...
@@ ...

	So you can chop sections out from the @@ downwards to the next section.
Thats how I prune the patches down at least. Then again, if that file is
already patched, it often makes sense to just re-generate that whole
patch with the change included.
