OpenOffice.org build:

        This package contains the desktop integration work for
OpenOffice.org, and a much simplified build wrapper, making an OO.o
build / install possible for the common man. It is a staging ground
for up-streaming patches to OO.o.

	This release we merged in PLD Linux support and some of the Ark Linux
patch-set.

Changes in this release:
    + bug fixes
	+ CONFIGURE_OPTIONS from environment (Arkadiusz)
	+ de-bashizing (Arkadiusz)
	+ misc build fixes (Arkadiusz, Chris, Rene, Michael)
    + features
	+ VBA (Michael)
	    + attribute parsing
	    + improved tests
	    + more VBA than StarBasic compatible
	+ KDE (Kendy)
	    + helper program integration
	    + splash / branding
	+ Novell (Internal)
	    + file-a-bug button
	+ PLD Linux support (Arkadiusz Miskiewicz)
	    + glibc compile fix
	    + splash/branding
	    + stlport 4.6.x compile
	+ Ark Linux patches (Bernhard Rosenkraenzer)
	    [ still underway ]
	    + branding

Package available from:
       
http://ooo.ximian.com/packages/OOO_1_1_1/ooo-build-1.1.55.tar.gz

	To find out more and/or get involved in OOo hacking see
http://ooo.ximian.com/ and/or subscribe to our OO.o development
mailing list at:
        http://lists.ximian.com/mailman/listinfo/openoffice.
