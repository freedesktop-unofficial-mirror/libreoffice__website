Hi,

you'll need to look at ooo-wrapper.in and package-ooo.  Unfortunately,
there is no way yet at runtime to change widget sets, since you have to
know the architecture of your platform, but also be root to relink the
widget set library to the normal OOo vcl library.  That's not a problem in
a personal install in any case.

Make sure you:

1) have <ooo-install-dir>/program/widgetsets/*/libvcl645*.so
2) have a widgetset activated.  see where your
<ooo-install-dir>/program/libvcl645*.so points too, make sure its pointing
to a widgetset-specific library from <ooo-install-dir>/program/widgetsets

If you don't have (1), go copy them from your
ooo-build/build/OOO_1_1_*/vcl/unxlngi4.pro* directories.

Dan
