Hi,

I removed everything older than m156 from the apply file & added support for 
oob680-m1.  Patches apply for both, you just have to choose the master 
workspace by specifying --with-tag=src680-m156 or --with-tag=oob680-m1.  I 
guess we'll branch at some stage, but for now - let's focus on 2.0.2, please.

Side note: It is possible to specify more MWSes in the apply file now, like 
the following:

[ TheSection < src680-m156 >= oob680-m3 ] will apply just for src680-m1 to 
src680-m155, and for oob680-m3 and higher.

[ TheSection < src680-m156 ] will apply for all oob680 milestones, keep it in 
mind in case you wanted to add support for e.g. m157 - but please do not do 
that now ;-)

Thank you,
Jan
