OpenOffice.org build:                                                                                                                      
                                                                                                                                           
        This package contains Desktop integration work for                                                                                 
OpenOffice.org, several back-ported features & speedups, and a much                                                                        
simplified build wrapper, making an OO.o build / install possible for                                                                      
the common man. It is a staging ground for up-streaming patches to                                                                         
stock OO.o.                                                                                                                                

                                                                                                                             
Changes in this release:                                                                                                                   
                                                                                                                                           
ooo-build-oob680.1.0

    + features:
        + target oob680-m1 [=2.0.2rc1]
        + initial support for qatesttool (Petr)
    + bug fixes:
        + lots fixes in the VBA stuff (Noel)
        + 'like' Evo2 handling (Michael)
        + Novell corporate dictionary (Michael)
    + build bits:
        + some fixes for parallel make (Giuseppe)
        + build with system db 4.3 fix (Bernhard, Hanno)
        + updated Ark, Gentoo, Novell bits (Bernhard, Andreas, Tor)
...

I am sorry. I forgot to announce the many ooo-build releases during last 
months. Please, look into ooo-build-oob680.1.0/NEWS too see what has happend.
