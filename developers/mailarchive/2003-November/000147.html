Dear all,
we finally got through our first build of OOo 1.1; with all bells and
whistles. Some comments:

* Interface fonts get too big; we had to revert a patch in
gui-font-scale.diff, in salgdi.cxx, and remove the part

-        if ( rDPIY < 96 )
-        {
-            rDPIX = Divide( rDPIX * 96, rDPIY );
-            rDPIY = 96;
-        }
-

To get reasonable sizes. I seem to remember that there was already a
discussion about it a while ago; I believe that Xft or X11 report 72 as
DPI, and without the above patch the font face is 30% larger.

* The pdf export button in the toolbar does the export with the screen
resolution for images by default. Can I suggest to set it to "press"? It
seems more logical; eventually, do anybody know where the default is set?

* Also: what is the license for the icons created by Ximian? We would like
to distribute our build of OOo with your icons; of course giving due
credit and the url to get patches and files.
Cheers
