> Hi Ruslan,

> On Fri, 2004-09-03 at 10:22 +0300, Ruslan wrote:
> > Could somebody tell me how to repeat OO build with ooo-build in case i
> > have already build OO and after that i added unzipped localized
> > helpcontent into $SRC_ROOT/solver/645/wntmsci8.pro/pck
>
>  This is really a question for Petr I think, he hacked that code.
>
> > Just running 'make' in the top of ooo-build begins to run and stops with
> > saying - nothing to do. I tried to delete compiled setup from instsetoo
> > and ran'make' again - result is the same - nothing to do.
>
>  Try removing some stamp/ files - those are used to mark whether the
> build has been completed etc.

In your case I suggest to remove the file ooo-build/stamp/build and run make 
again.

Another solution would be to put the tar-ball with localized help content into 
the directory ooo-build/src/, for example 
ooo-build-1.3.2/src/helpcontent_49_unix.tar.bz2. Then it will be installed 
automatically within "make install". See ooo-build/bin/install-help for more 
details.
