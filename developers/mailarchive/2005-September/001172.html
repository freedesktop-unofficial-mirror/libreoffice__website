side affect. If the patch is applied to an existing ooo-build tree ( i.e 
one thats built already and you've just updated and applied latest 
patches ). The observable affect is you will get a core and a message 
about a uncaught RuntimeException on startup. To get rid of this just do 
a clean build in stoc, don't ask why, I don't know, ya just gotta do it ;-)
