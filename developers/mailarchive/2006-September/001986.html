The process looks good. 

The localization of OOo is defined by locales by default. Could you please 
cut&paste the output from the following command?

	locale

You could try to start OOo using the following command:

LANG=sw_KE.UTF-8 /opt/OOInstall/program/soffice

Hmm, it seems that the Swahili locales sw_KE.UTF-8 are not supported by linux 
system. Well, you could also set the Swahili localizations in menu in 
"Tools/Options.../Language Settings/Languages/User Interface". You would need 
to restart OOo after you change the UI language.

If it does not help, could you check if your installation includes the files 
(ooo-installation-prefix)/program/resource/*sw.res?
