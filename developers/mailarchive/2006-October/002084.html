This is where all the symlinks end up, when i specify
UNO_JAVA_JFW_JREHOME=file:///usr/lib/jvm/java-gcj. When I leave the Java
framework bootstrap variables at the default, JAVA_HOME (set to /usr by
the build system)  is used to determine the JRE path, and the java
executable is set to /usr/bin/java. On Mandriva 2007 /usr/bin/java is
linked to jamvm (via /etc/alternatives/java), which by the way causes
exactly the same error messages as gij.
