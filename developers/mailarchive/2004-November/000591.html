Mandreiana (@openoffice.org) suggested me to submit to submit the following
proposal to this list.

Integrate native support for IM (Kopete on KDE - Gaim on Gnome)
Office messaging, similar to send document by mail but that it can call on your
kopete online buddies. This will make easy exchange of documents through kopete
directly from the OOo interface.

So it would be something like:
-File
   - Send
     - Kopete
       - [online buddy list]

--------
some concerns
- might be slow
- might create dependencies for OOo

--------
some alternatives
add-on or plugins that call to the .gaim/ or .kopete/ xml files containing this
buddy list and also call the command to send the file.

-------
Why do this?
Increase user experience as some files are not saved for that reason you cant
really do it from nautilus. (I barely save my files when I use send by e-mail)
