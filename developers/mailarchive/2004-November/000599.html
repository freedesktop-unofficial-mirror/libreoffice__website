	This is in fact a feature; previously the build would build tens of
huge install sets in there.

	2 ways to fix it:

	a) hack bin/build-ooo and change BUILD_MINIMAL to your own 
	   language telephone code.

	b) cd instsetoo/util ; dmake pack_<code>
