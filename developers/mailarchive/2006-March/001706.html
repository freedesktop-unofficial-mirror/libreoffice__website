> Also does it exists an internal font replacement table? For instance I've
> seen that when OOo imports a document which use the "Times New Roman"
> fonts, automatically at screen the "Vera Serif" is used as replacement
> (since Times New Roman is generally not available on Linux), but on the
> other hand I do not see the "Vera Serif" listed in the replacement table
> of menu Tools/Fonts/Replacement Table.

Most of the fonts are configured in 
<OOO_HOME>/share/registry/data/org/openoffice/VCL.xcu

This file has two parts. The first one list preferred fonts for various 
localizations and various situations, like a font for menu (display), 
heading, ... Note that the font for menu might be read from KDE or GNOME if 
you use the related vcl plugin.

The second part of VCL.xcu defines default replacements for various known 
fonts. The item SubstFonts is used for display. The item SubstFontsPS is used 
for export to postscript, ...

Finally, the fonts for help content are configured in the various CSS files in 
<OOO_HOME>/help/<lang>/*.css.
