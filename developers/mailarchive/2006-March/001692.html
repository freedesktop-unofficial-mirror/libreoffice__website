Hi,

OOo-2.0.2 is officially announced. Here are my plans to release 
ooo-build-2.0.2:

1. I have packaged OOO_2_0_2 sources over night. There should at least one 
more master fix in compare to the oob680-m5 sources. I plan to check and 
upload the tarballs them within an hour or two.

2. I am going to remove the old stuff from ooo-build HEAD, so it will support 
only oob680-m5 at the moment.

3. Then I'll add support for OOO_2_0_2 build tag. It should be ready within 
2-3 hours, about 11a.m. UTC.

4. Then I'll start a test build.

5. If it succeed, I'll release first version of ooo-build-2.0.2. I expect that 
it will happen today evening, about 5p.m. UTC.


Note 1: We expect that everybody wants to work on the OOO_2_0_2 build, so we 
do not plan to branch ooo-build-2-0-2 now. The branch will be announced when 
it happens sometimes later.

Note 2: I could release ooo-build-2.0.2.x more times, for example on Friday, 
if a crytical bug is found in todays release...
