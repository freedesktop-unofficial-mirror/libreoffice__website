These are my cryptic notes on the subject which should get you going.
Hope they make some sense

David

splash screen
  offmgr/res/
    openabout.bmp - help/about
    openintro.bmp - intro screen

productname
  # this seems also to contain the name of the bitmaps used in about/splash
  offmgr/source/offapp/intro
    ooo.src
  # default install path takes whole application name so override it in 
config.scp
  # this seems also to contain the name of the bitmap used in 
installation (oo_product.bmp)
  scp/source/office/config.scp
  * to include the bitmap, also need to change scp/source/office/files.scp

license is in instset2/util/openoffice.lst

installation background in readlicense_oo/source/cinema
  oo_product.bmp
