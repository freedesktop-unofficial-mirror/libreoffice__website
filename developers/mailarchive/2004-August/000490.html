Hi Ruslan,

On Thu, 2004-08-19 at 00:26 +0300, Ruslan wrote:
> Is it possible to continue build(using ooo-build) from the place, where
> error occured(after fixing) and not from the very beginning?
> As i was told on #openoffice.org i need to run  build --all:<project>
> from 'instsetoo', where project is the name of the project where i got
> error. What steps are needed to make this with ooo-build?

	Yes; with a recent ooo-build; such as 1.3.0 you can easily do that
[ you should be using that on Win32 too - certainly ]. Just type 'make'
again at the toplevel.

	It re-does some work, such as re-configuring, and random parts of the
build seem to insist on re-building gratuitously; but it does do an
incremental build.
