OpenOffice.org build:

	This package contains Desktop integration work for
OpenOffice.org, several back-ported features & speedups, and a much
simplified build wrapper, making an OO.o build / install possible for
the common man. It is a staging ground for up-streaming patches to
stock OO.o.

	This is a brown-paper-bag release; a couple of acute build sillies, and
a mistake in the release announcement, plus a few nasty bugs nailed.

Changes in this release:

    + bug fixes
	+ VFS URI mangling (Michael)
	+ gnome file-sel bugs (Raul)
	+ KDE file-sel 'cancel' crasher (Jan)
	+ print-resolution fix (Petr)
    + features
	+ OOO_FORCE_DESKTOP= (Tommy C. Li)
	+ install more hyphen dicts (Petr)

Package available from:
       
http://ooo.ximian.com/packages/OOO_1_1_2/ooo-build-1.3.2.tar.gz

	To find out more and/or get involved in OOo hacking see
http://ooo.ximian.com/ and/or subscribe to our OO.o development
mailing list at:
        http://lists.ximian.com/mailman/listinfo/openoffice.
