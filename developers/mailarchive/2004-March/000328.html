Salvatore Enrico Indiogine wrote:
> Please, what is the name of the Debian 1.1.1 OOo package so that I can use:
> 
> apt-get install <OOo 1.1.1>
> 
> I am a Debian newbie trying to migrate from RedHat.

I assume you are using stable? There is no 1.1.1 package backported to stable
yet.

There is a 1.1.1rc3 package (with broken inter-package dependencies,
though) in unstable which afais will work on testing too. But we don't
have one for stable yet.

There is a woody-_test_ backport of 1.1.0 at

#debian OOo test backport (i386 only)
deb http://ftp.freenet.de/pub/debian-openoffice/ woody-test main contrib

please note that this is not called _test_ for fun, there were some font
problems there...

otherwise he have a 1.0.3 backport ;) and an openoffice.org 1.1beta2
backport which I think you do not want :-)
