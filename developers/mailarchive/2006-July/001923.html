I just disabled it manually for my builds in solenv/inc/unxlngi6.mk, by 
changing CFLAGSWERRCC=-Werror into CFLAGSWERRCC=. Then I reenable it when I 
need to compile my own code.

issues in other people's code (just take the .mk file that matches your build 
platform).
