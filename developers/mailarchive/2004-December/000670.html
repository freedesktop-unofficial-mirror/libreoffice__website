On Fri, 2004-12-24 at 14:41 +0100, Hanno Meyer-Thurow wrote:
> Anyone may explain me how to package the compiled source not using
> epm? Thanks a lot!

	For an RPM based system, you need to read bin/package-ooo - at some
stage it moves the RPM generated files splits, and builds a few file
lists in $OOBUILDDIR AFAIR.

	Ultimately - we can't use EPM packaging in the SuSE or RH builds, so -
we just use the result of 'make install' in the toplevel; to do that you
prolly want to configure ooo-build with --disable-postinstall-scripts to
stop some user-mode install scripts running. Also, post-install you want
to do them in your package; check-out what that flag does in
bin/package-ooo to see what to do.
