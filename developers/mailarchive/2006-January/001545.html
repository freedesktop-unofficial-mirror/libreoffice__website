Currently our PO files do not allow translators to give us feedback
about bad English strings.  The msgid-bug-address feature of Gettext
allows them to easily find out where to send corrections or ask
questions.

This patch adds that ability.

1) intltoolize --copy --force --automake
2) apply the attached patch (GNOME bug at
http://bugzilla.gnome.org/show_bug.cgi?id=329346 )

Now we're ready

1) ./autogen.sh
2) cd po
3) ../intltool-update --pot

This email list is now in the bug msgid bug address in the POT file.
