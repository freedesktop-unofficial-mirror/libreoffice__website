Giuseppe found that there are some problems with galleries in OOo-2.0.1. I 
reproduced the same problems with OOO_2_0_1 sources and ooo-build-2.0.1.

The problems are:

1. We use ooo-build/bin/build-galleries script to generate extra galleries
