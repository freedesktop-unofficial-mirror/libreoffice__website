I have added a possibility to install OOo below /usr/lib64 into ooo-build HEAD 
yesterday. It is useful on 64-bit bit biarch systems like on x86_64 when you 
build the 64-bit package. It is enough to define the configure switch:

	./configure --libdir=/usr/lib64
