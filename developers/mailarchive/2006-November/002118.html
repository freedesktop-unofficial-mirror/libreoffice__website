the first chunk from this patch is not applied properly on Windows.

    ...
    SB_DLLPRIVATE void   AddDimImpl32( INT32, INT32, BOOL bAllowSize0 );
+   bool mbHasFixedSize;

which is applied for Linux systems only. On Windows, basic/inc/sbx.hxx
has AddDimImpl32 without SB_DLLPRIVATE.

I found this in stable 2.0.4 branch and I didn't check this in HEAD yet.
