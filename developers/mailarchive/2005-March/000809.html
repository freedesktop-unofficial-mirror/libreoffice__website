	Rene's nice libwpd backport changes enough that an incremental build
fails. If you update to HEAD ooo-build from an older version you'll need
to do:

	rm -Rf build/src680-m79/solver/680/unxlngi4.pro/inc/libwpd
	rm -Rf build/src680-m79/writerperfect/unxlngi4.pro
