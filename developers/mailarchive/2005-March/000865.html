Hi,

I'm running Debian Linux and there the (Open)Office Menu looks more or 
less like this. ( The icons got updated to ximian ones lately)

http://shots.osdir.com/slideshows/slideshow.php?release=184&slide=63

I personally do not find the long names like OpenOffice.org From 
Template aesthetically very beautiful. Furthermore also the comments
are not really helpful comment for OpenOffice.org Math is if I recall 
correctly something like OpenOffice.org Math Formula

My suggestion is that ooo-build will include the desktop files i.s.o. 
every distro making its own. Furthermore I'd suggest to change the names 
and comments so a novice user can get started more easily.

Here are my suggestions

Writer
Name=OOo Word Processor
Comment=Write and Read Documents

Calc
Name=OOo Spreadsheet
Comment=Create and View Spreadsheets

Draw
Name=OOo Draw
Comment=Create and View Drawings

 From Template
Name=OOo From Template
Comment=Create a new OOo Document from a Template

Impress
Name=OOo Presentation
Comment=Create and View Presentations

Math
Name=OOo Formula Editor
Comment=Create and Edit Formulas

Writer/Web
Name=OOo HTML Editor
Comment=Create and Edit Web Pages

Printer Adminstration
Name=OOo Printer Administration
Comment=Configure your Printer for use in OOo
