Ok I got past the part of my last few emails but now it is stopping at a
diffrent point that I will port later in the email.  Another thing I
found useful while compiling gcc if you add these flags to the compile
it will compile faster and will not compile java which you do not need.

--enable-static --disable-shared --enable-languages=3Dc++

That way you only build gcc and g++ and also it makes it so for those of
us who have tar as a dynamic link can compile the gcc part of it.  I
noticed if I compiled tar with static links it would compile gcc but if
I compiled tar with dynamic links it would not build gcc.  Now I have no
idea why but this fixed it. =20
