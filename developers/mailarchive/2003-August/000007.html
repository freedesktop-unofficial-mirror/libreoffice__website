Every now and then Debian unstable is Debian broken... This time it
looks like libcupsys2-dev requires libgnutls5-dev and libgnomevfs2-dev
requires libgnutls7-dev.

( http://bugs.debian.org/cgi-bin/bugreport.cgi?bug=203059 )

You could wait for libcupsys to be updated or fetch older libgnomevfs2
packages from testing (2.2.5-2) (I don't know if you have to downgrade
other gnome packages to install them).

I can send a patch for the official Debian OOo packaging scripts and
some instructions on how to build Ximianized packages, if you want that,
jorge.
