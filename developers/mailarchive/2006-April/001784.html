Yes, sure :o)

OpenOffice.org Premium is mainly an integration project. So the extras 
are packed into OpenOffice.org installer. You do not need to install, 
copy, set other files, all stuff is integrated in Premium. Of course 
from media point of view Premium mainly contains Hungarian language 
templates and more extended international fonts (additional bitstream 
vera replacement to deja vu is on the way....), galleries, fontwork 
gallery (already in the main stream)... I would like to make 
OpenOffice.org project more international, so we released en-US Premium 
based on 2.0.2 and we will release HU, en-US, and de-DE based on 2.0.3.
For more information about OpenOffic.org Premium is here:
http://www.openoffice.org/servlets/ReadMsg?list=dev&msgNo=16407

http://hup.hu/node/23206
