It is related to the configure switch --without-myspell-dicts. It is used by 
many distributions but not by all, see ooo-build/distro-configs.

I have updated ooo-build/download.in (in cvs HEAD), so that it will download 
-system.tar.bz2 for all disctributions that do not use 
--without-myspell-dicts in ooo-build/distro-configs (including Frugalware).

The list is still hardcoded there. It would be better to join it will the 
--without-myspell-dicts configure check. I'll most probably do it later as 
part of upstreaming the ooo-build stuff.
