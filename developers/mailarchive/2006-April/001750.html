I am going to move config_office-ldap.diff to BuildBits now.

It defaults to build LDAP connectivity via Mozilla's LDAP.

To disable it, use:
	--disable-ldap
	--disable-mozilla does it as well

To use OpenLDAP, use:
	--with-openldap

--with-openldap is set in distro-configs/Common.conf.in.
You may stop using the hacks in ooo-build.
