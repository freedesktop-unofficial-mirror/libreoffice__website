The download/unpack phase is skipped in incremental builds. The next clean 
build should be without problems.

If you want to continue the current build, you can:

1. download:

	http://go-oo.org/packages/libwpd/libwpd-0.8.8.tar.gz
	http://go-oo.org/packages/SRC680/libwps-0.1.0~svn20070129.tar.gz

2. Install them at:

	ooe680-m6/libwpd/download/libwpd-0.8.8.tar.gz
	ooe680-m6/libwps/download/libwps-0.1.0~svn20070129.tar.gz

3. I would also remove the current build dir in the libwpd module 

	rm -rf 	ooe680-m6/libwpd/unxlngi6.pro

Then it should build fine...
