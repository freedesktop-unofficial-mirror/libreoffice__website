something like:
	make DESTDIR=$RPM_BUILD_ROOT install
or
	export DESTDIR=$RPM_BUILD_ROOT
	make install

If I remember correctly, the file lists are not generated if the DESTDIR is 
not set. This might be the problem.

We should probably fix "make install", so it works even without DESTDIR set.
