As many of you have seen, the go-oo wiki has been defaced.  Because of
the versioning nature of wiki, it was simple to revert the changes
(kudos to Kendy and Henrycruther).  I have temporarily blocked the
offending user.  If he is determined, he will return and there is only
so much we can do to stop him.  New users can be added by simply coming
up with a new identity.  IP numbers and rangers can be blocked, but in
the world of dynamic IP's, it's a crapshoot.

Defacement is the price of openness, I'm afraid.  We have a few options:
1) protect the main page, which will only allow admins to edit it.  This
will make defacement less attractive, because it won't be on the main
page.

2) make new users moderated, in that the admins will have to create user
accounts by request

3) be vigilant and simply revert changes as they happen

Your thoughts appreciated.
