IIRC, changing isZeroBit in hsqldb/src/org/hsqldb/HsqlDateTime.java to
something like

        private boolean isZeroBit(int bit) {
            if (state & (1L << bit))
                return false;
            return true;
        }

will work around that problem, someone should probably log a bug against
the gcc bugzilla about it.
