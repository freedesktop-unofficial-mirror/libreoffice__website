> I would like to ask which parts of ooo-build depend on ImageMagick and
> if it is possible to get rid of that dependancy ?

The original 26x24px toolbar icons have to be converted to 24x24px to be
used together with the new icons. ooo-build/bin/scale-icons does that
using convert from ImageMagick. Some . You'd have to hack out all of the
new-icons patches/build system scripts to do a build without
ImageMagick.
