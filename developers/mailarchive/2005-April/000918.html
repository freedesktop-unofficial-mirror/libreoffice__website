Hey,

You should be using ooo-build/bin/ooinstall <path-to-install>. The make =
install was really a 1.1.x thing. Also everytime after u rebuild something =
you need to run "deliver" as this moves the built libraries to the solver/ =
directory as the install script picks up libraries from solver when it =
installs.

Ofcourse you could by pass this all by using bin/ooinstall -l <path-to-inst=
all> which nicely links up your build and installation trees.

p.s: Also look out for solenv/bin/linkoo if the -l option fails.
